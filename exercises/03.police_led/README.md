# Exercício 03 – LEDs Tipo Polícia

## Objetivo
Aprender a controlar múltiplos LEDs e aplicar lógica de alternância para simular luzes de polícia.

## Componentes
- Arduino / ESP32
- LED vermelho
- LED azul
- 2 Resistências (≈ 220Ω)
- Breadboard
- Cabos jumper

## Funcionamento
Dois pinos digitais são configurados como saída, um para cada LED.  
No `loop()`, os LEDs são ligados de forma alternada: enquanto um LED está ligado, o outro permanece desligado, criando o efeito visual de luzes de polícia.

## Observações
Este exercício ajudou a compreender o controlo de vários pinos digitais e a aplicação de lógica simples de alternância.
